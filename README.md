# Point Planner

HMVスペシャルクーポン利用と通常クーポンを含めて、同一商品の分割購入プランを試算するツールです。

Web UI は `web/`、CLI は [`point_planner.py`](./point_planner.py) を使います。

## 前提

- 商品は同一税込価格
- スペシャルクーポンは `1円 = 1円分`
- スペシャルクーポン付与は利用後金額を元に計算
- 通常クーポンは「税込み注文金額がしきい値以上なら値引き」
- 通常クーポンは複数種類・複数枚を保持可能
- 1注文につき使える通常クーポンは1枚まで
- HMVスペシャルクーポンと通常クーポンは同時利用しない前提

## CLI

```bash
python point_planner.py 12 --start-points 3000 --coupon 10000:500x2 --coupon 15000:1000x1
```

### 主な引数

- `n`: 購入枚数
- `--start-points`: 開始時スペシャルクーポン残高
- `--unit`: 1枚の税込価格
- `--tax`: 税率（%）
- `--rate`: スペシャルクーポン還元率（%）
- `--min`: スペシャルクーポン付与最低金額
- `--basis`: スペシャルクーポン付与条件の判定基準 (`order_total` / `cash`)
- `--coupon`: 通常クーポン定義。`10000:500x2` は「税込み1万円以上で500円引き」を2枚
- 計算結果には、現金支払を1枚分以内だけ増やした場合に追加購入できる枚数のサジェストも表示されます。

## Web

```bash
cd web
cmd /c npm run dev
```

詳細設定の `所持通常クーポン` に、1行ごとに `10000:500x2` 形式で入力します。
